# Hazard Situation Analysis and Prediction
This project is a web-based platform developed at Lapland UAS for analyzing and visualizing dangerous driving situations using sensor data.
The system is built around truck driver hazard button presses, which indicate moments of risk on the road. These events are combined with sensor and environmental data to better understand and predict hazardous condition.
Deployment Limitations:
Not all project components are included in the deployed website.
Specifically: The **Route Risk Analysis module** is not available online, as it includes files that cannot be public.
