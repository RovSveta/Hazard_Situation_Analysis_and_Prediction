# Hazard Situation Analysis and Prediction
This project is a web-based platform developed at Lapland UAS for analyzing and visualizing dangerous driving situations using sensor data. The system is built around truck driver hazard button presses, which indicate moments of risk on the road. These events are combined with motion and environmental sensor data to better understand and predict hazardous conditions.

The project includes a website that presents the results in an interactive way. The website features an interactive map where users can explore hazard locations, based on data collected from 18 button press events. It also provides a summary of the project outcomes and insights from the dataset.

In addition, the website contains a page with a Route Risk Map based on Machine Learning (ML Model 2), which visualizes route-level risk analysis.

The chatbot (ML Model 1) is developed as a separate component and is not integrated into the website. It is used independently for hazard analysis.

Limitations: Not all project components are included to this repo. The Route Risk Analysis module (ML Model 2) is not available online, as it includes files that cannot be public.
